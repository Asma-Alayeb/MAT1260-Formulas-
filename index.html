MAT1260 Formula Sheet

Descriptive Statistics

  • Relative frequency:\frac{ frequency} {sum of all frequencies}\
  • Class midpoint:
    \((\text{lower class limit}_i + \text{lower class limit}_{i+1}) / 2\)
  • Mean:
    \(\mu = \frac{\sum x_i}{n}\)
  • Population standard deviation:
    \(\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}}\)
  • Sample standard deviation:
    \(s = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n-1}}\)
  • Population variance:
    \(\sigma^2 = \text{Var.P(numbers)}\)
  • Sample variance:
    \(s^2 = \text{Var.S(numbers)}\)

Z-scores and Quartiles

  • Z-score:
    \(z = \frac{x - \mu}{\sigma}\)
  • Find \(x\) from \(z\):
    \(x = \mu + z \sigma\)
  • Quartiles:
    \(Q_1 = \text{QUARTILE.INC(numbers,1)}\)
    \(Q_2 = \text{QUARTILE.INC(numbers,2)}\)
    \(Q_3 = \text{QUARTILE.INC(numbers,3)}\)
  • IQR:
    \(\text{IQR} = Q_3 - Q_1\)
  • Fences:
    \(\text{Lower fence} = Q_1 - 1.5 \cdot \text{IQR}\)
    \(\text{Upper fence} = Q_3 + 1.5 \cdot \text{IQR}\)
  • Five-number summary: Min, Q1, Q2, Q3, Max

Correlation and Regression

  • Correlation coefficient:
    \(r = \text{CORREL(array1, array2)}\)
  • Regression line:
    \(y = mx + b\)
  • Slope:
    \(m = \text{SLOPE(ys,xs)}\)
  • Intercept:
    \(b = \text{INTERCEPT(ys,xs)}\)
  • Residual:
    \(\text{Residual} = \text{Observed value} - \text{Predicted value}\)

Probability Rules

  • General Addition Rule:
    \(P(A \text{ or } B) = P(A) + P(B) - P(A \text{ and } B)\)
  • Complement Rule:
    \(P(A^c) = 1 - P(A)\)
  • Multiplication Rule:
    \(P(A \text{ and } B) = P(A) \cdot P(B|A)\)
  • Conditional Probability:
    \(P(A|B) = \frac{P(A \text{ and } B)}{P(B)}\)

Binomial Distribution

  • Mean:
    \(\mu_{bi} = n \cdot p\)
  • Standard deviation:
    \(\sigma_{bi} = \sqrt{n \cdot p \cdot (1-p)}\)
  • Excel: BINOM.DIST(x, trials, probability, FALSE)

Normal Distribution

  • Area to the left:
    \(P(X \le x) = \text{NORM.DIST}(x, \mu, \sigma, \text{TRUE})\)
  • Area to the right:
    \(P(X > x) = 1 - \text{NORM.DIST}(x, \mu, \sigma, \text{TRUE})\)
  • Area between:
    \(P(a \le X \le b) = \text{NORM.DIST}(b, \mu, \sigma, \text{TRUE}) - \text{NORM.DIST}(a, \mu, \sigma, \text{TRUE})\)
  • Given percentile, find number:
    \(x = \text{NORM.INV}(\text{area}, \mu, \sigma)\)

Sampling Distributions

  • Sample mean:
    \(\mu_{\bar{x}} = \mu\)
  • Std dev of sample mean:
    \(\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}}\)
  • Sample proportion:
    \(\hat{p} = \frac{x}{n}\)
  • Mean of sample proportion:
    \(\mu_{\hat{p}} = p\)
  • Std dev of sample proportion:
    \(\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}}\)
