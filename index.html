MAT1260 Formula Sheet

Descriptive Statistics

  • Relative frequency=\(\frac{frequency}{\text{sum of all frequencies}}\)
  • Class midpoint=\(\frac{\text{lower class limit + Upper class limit}}{2}\)
  • Mean:
    \(\mu = \frac{\sum x_i}{n}\)
  • Excel Mean: =average(numbers)
  • Population standard deviation:
    \(\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}}\)
  • Excel population standard deviation =stdev.p(numbers)
  • Sample standard deviation:
    \(s = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n-1}}\)
  • Excel sample standard deviation =stdev.s(numbers)
  • Population variance:
    \(\sigma^2\) = VAR.P(numbers)
  • Sample variance:
    \(s^2\) = VAR.S(numbers)

Z-scores and Quartiles

  • Z-score:
    \(z = \frac{x - \mu}{\sigma}\)
  • Excel z-score =STANDARDIZE(X, mean, standard_deviation)
  • Find \(x\) from \(z\):
    \(x = \mu + z \sigma\)
  • Quartiles:
    \(Q_1\) =QUARTILE.INC(numbers,1)
    \(Q_2\) =QUARTILE.INC(numbers,2)
    \(Q_3\) =QUARTILE.INC(numbers,3)
  • IQR:
    \(\text{IQR} = Q_3 - Q_1\)
  • Fences:
    \(\text{Lower fence} = Q_1 - 1.5 \cdot \text{IQR}\)
    \(\text{Upper fence} = Q_3 + 1.5 \cdot \text{IQR}\)
  • Five-number summary: Min, Q1, Q2, Q3, Max

Correlation and Regression

  • Correlation coefficient:
    \(r\) = CORREL(array1, array2)
  • Regression line:
    \(y = mx + b\)
  • Slope:
    \(m\) = SLOPE(ys,xs)
  • Intercept:
    \(b\) = INTERCEPT(ys,xs)
  • Residual:
    \(\text{Residual} = \text{Observed value} - \text{Predicted value}\)

Probability Rules

  • General Addition Rule:
    \(P(A \text{ or } B) = P(A) + P(B) - P(A \text{ and } B)\)
  • Complement Rule:
    \(P(A^c) = 1 - P(A)\)
  • Multiplication Rule:
    \(P(A \text{ and } B) = P(A) \cdot P(B|A)\)
  • Conditional Probability:
    \(P(A|B) = \frac{P(A \text{ and } B)}{P(B)}\)

Binomial Distribution

  • Mean:
    \(\mu_{bi} = n \cdot p\)
  • Standard deviation:
    \(\sigma_{bi} = \sqrt{n \cdot p \cdot (1-p)}\)
  • Excel: =BINOM.DIST(x, trials, probability, FALSE)

Normal Distribution

  • Area to the left:
    \(P(X \le x)\) =NORM.DIST(x, Mean, Standard Deviation, TRUE)
  • Area to the right:
    \(P(X > x)\) = 1 - NORM.DIST(x, Mean, Standard Deviation, TRUE)
  • Area between:
    \(P(a \le X \le b)\) = NORM.DIST(b, Mean, SD, TRUE) - NORM.DIST(a, Mean, SD, TRUE)
  • Given probability (area), find number:
    \(x\) = NORM.INV(area, Mean, Standard Deviation)
